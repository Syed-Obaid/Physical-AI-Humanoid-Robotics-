  - [module-6/](./img/module-6/) - Images for Module 6: Final Project

## Image Guidelines

- Use descriptive filenames that explain the content
- Include alternative text descriptions for accessibility
- Keep image sizes optimized (under 500KB per file)
- Use SVG format for diagrams when possible
- Use WebP or optimized JPG/PNG for photographs
