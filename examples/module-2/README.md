# Example Spec-Kit Configuration

This directory contains example configuration files and scripts for using Spec-Kit Plus in the AI/Spec-Driven Book Creation process.

## Files

- `.specify/memory/constitution.md`: The project constitution that defines the principles and governance
- `.specify/templates/spec-template.md`: Template for feature specifications
- `.specify/templates/plan-template.md`: Template for implementation plans
- `.specify/templates/tasks-template.md`: Template for task breakdowns
- `.specify/scripts/bash/`: Various utility scripts for project management

## Usage

This configuration demonstrates how to set up a Spec-Kit project for book creation:

1. Initialize a new project with the provided templates
2. Define your book's constitution in the memory folder
3. Use the templates to create specs, plans, and tasks
4. Execute the workflow as defined in the scripts
