# Sample Claude Code Prompts for Book Creation

This directory contains sample prompts that can be used with Claude Code for AI-assisted book creation.

## Files

- `book-outline-prompt.txt`: Prompt for generating a book outline from specifications
- `chapter-draft-prompt.txt`: Prompt for drafting a chapter based on specifications
- `content-enhancement-prompt.txt`: Prompt for improving existing content
- `technical-review-prompt.txt`: Prompt for technical accuracy review
