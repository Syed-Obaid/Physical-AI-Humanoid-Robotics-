# Chapter Drafting Prompt

You are a technical writing expert helping to draft a book chapter based on the following specifications and outline:

## Specifications
[INSERT CHAPTER SPECIFICATIONS HERE]

## Outline

## Task
Create a full chapter draft following the structure:
1. Learning Objectives (specific and measurable)
2. Prerequisites (what reader needs to know)
3. Content (Introduction → Core Concepts → Examples → Applications)
4. Summary (key points covered)
5. Exercises (at least 3: conceptual, computational, implementation)
6. References (APA format, minimum 3 sources)

## Requirements
- Write in clear, accessible language
- Use specific, practical examples
- Include at least one code example if relevant (with language specification)
- Follow educational clarity best practices
- Ensure technical accuracy
- Maintain consistency with terminology in docs/glossary.md

## Constraints
- Do not exceed 2000 words
- Use second person ("you will learn") for tutorials
- Use third person ("the system computes") for theory
- Include proper APA citations for all technical claims
